## Wdrożenie
1) Wgraj do repo jako root.  
2) Netlify: Build: brak, Publish: `.`, Functions: `netlify/functions`.  
3) Zmienna: `GROQ_API_KEY` z https://console.groq.com/keys  
4) Deploy.  
5) UI pozwala wybrać tryb, temperaturę i max tokens.
